# _Beelzebub v1.3_
Beelzebub Shell Scanner

<img src="https://github.com/InMyMine7/InMyMine7/blob/main/1.3.png">

Feature : 
- Fast Scanning
- Multithread
- Support http/https
- Safe for use no backdor or etc
- zone h grabber (change ppshid in zoneh.json)
- domain cleaner
- reverse ip

Requirement :

- `Python 3.11.4`

- `pip 23.1.2`

Install :

```
pip install -r beelzebub.txt
```
How To Usage :

```
python beelzebub.py
```
